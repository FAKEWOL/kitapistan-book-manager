# Kitapistan Web Application

## Description
This is a simple web application built with PHP to manage book data. It connects to a MySQL database to store and retrieve book information.

## Features
- Add new books
- View existing books

## Installation
1. Clone the repository
2. Set up a local server environment (e.g., XAMPP, WAMP)
3. Create a MySQL database named `DersDB`
4. Import the provided SQL file to create the necessary tables
5. Configure the database connection in `index.php`

## Usage
1. Open your web browser and navigate to the application's URL
2. Use the form to add new books
3. View the list of books stored in the database

## Tech Stack
- PHP
- MySQL
- HTML/CSS
